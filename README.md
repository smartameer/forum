# GQL Forum

### Installation
    yarn install

### DB setup
    npx prisma migrate

### Development
    yarn dev

### DB viewer
    npx prisma studio


## Details
---
This will run a forum server with
  - Posts
    - Create/Read/Update/Delete Post
    - Comments
      - Create/Read/Update/Delete Comment
      - Votes
        - Add/Remove Vote
    - Votes
      - Add/Remove Vote
  - Users
    - Login
    - Registration
    - Account Updation
